## How to Use This Log

As you work through Fullspecs on a real project, log anything that:
- Slows you down
- Feels unnecessary
- Is confusing
- Is missing
- Doesn't match reality
- Could be simpler

**Don't fix the methodology mid-project.** Just log it here. Fix after project completes.
