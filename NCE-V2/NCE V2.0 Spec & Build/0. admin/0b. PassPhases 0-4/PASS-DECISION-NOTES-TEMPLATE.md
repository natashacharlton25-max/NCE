# Pass Decision Notes

## Entry Format

```markdown
### {{timestamp}} — Phase {{number}}

**Type:** NOTE | ISSUE | DECISION | RESTART | RESOLUTION  
**Author:** Human | Claude  
**Phase:** {{phase number and name}}

**Context:**
What was happening when this occurred.

**Observation / Issue:**
What was noticed or what failed.

**Decision:**
What was decided.

**Action Taken:**
What was done.

**Outcome:**
Result after action.

**Restart Triggered:** Yes / No  
If Yes: Restarted at Phase {{number}}
```
