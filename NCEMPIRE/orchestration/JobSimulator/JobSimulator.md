> **Purpose:** Estimate cost and time before execution
> **Type:** Orchestration Module
> **Status:** Placeholder
> **Version:** 0.0.0

## Overview

JobSimulator runs a "dry run" before batch operations. Calculates prompt token size, estimates cost and time.

---

## Notes

### Role
- Run dry runs before batches
- Calculate token estimates via TokenManager
- Provide estimated cost and time
- Validate before user clicks "Go"

### Owns
- Simulation logic
- Cost estimation
- Time estimation
