## Overview

RulesEngine evaluates brand, content, AI, and publishing rules. Enforces constraints consistently and acts as single source of truth for "is this allowed?"

---

## Notes

### Role
- Evaluate brand, content, AI, and publishing rules
- Enforce constraints consistently
- Act as single source of truth for "is this allowed?"

### Owns
- Rule evaluation logic
- Rule prioritisation
- Pass / fail / warn outcomes

### Does NOT Do
- Store rules (libraries do)
- Fix content (checks/review do)
